# North Su

This repository contains code for analyzing the hydroacoustic data that captured the
2015 North Su submarine volcanic eruption.

## Files

[hdf5.ipynb](https://github.com/tjcrone/northsu/blob/master/hdf5.ipynb)

  Example Jupyter notebook (Python) showing how to load the HDF5 file released
  as part of the North Su submarine volcanic eruption analysis.
